# mad-science-scraper
What do you do when you can't do mad science? You scrape it.

# [Manual] Usage
 - Rename `config.yml.example` to `config.yml`
 - Fill in your values for `config.yml`
 - Run `src/scraper.py`
 - Let the scraping commence

# [Docker] Usage
 - Use the example `docker-compose.yml` file to run a container.
 - Use `SCRAPE_INTERVAL` to adjust the interval between script runs.
